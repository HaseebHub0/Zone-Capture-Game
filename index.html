Zone Clash Arena

This project is now set up for real room-based online multiplayer, where friends join from their own phones using a room code.

Important: online multiplayer cannot run from file:// mode. Start the local server first, then open the HTTP URL.

Run: node server.js

Then open: http://localhost:3000

The actual multiplayer client is served from the Node server under public/index.html, with room create/join flow and realtime synced gameplay.
